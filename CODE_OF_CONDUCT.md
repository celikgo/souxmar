# Code of Conduct

The souxmar project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct, in full and without modification. The canonical text is the authoritative version; this file points to it rather than copying it so that upstream amendments take effect automatically.

## Scope

The Code of Conduct applies to:

- All project spaces: the GitHub repositories, issue trackers, pull requests, discussions, official Discord and Slack channels, mailing lists, in-person and virtual events.
- All representations of the project in public, including by maintainers, contributors, and users acting in a project capacity.
- Both technical and non-technical interactions.

## Reporting

Report concerns to **conduct@souxmar.dev** (placeholder address; will be activated when the GitHub organisation is provisioned in Sprint 0).

Reports are handled by a rotating subset of maintainers — never by the maintainer who is the subject of a report. The reporting maintainer recuses themselves on receipt and routes the report to colleagues.

What to include in a report:

- What happened, when, where (link to the message / issue / PR if applicable).
- Who was involved.
- Any context that may help us understand the situation.
- Whether you would like the report kept confidential.

We acknowledge every report within 72 hours. We aim to resolve every report within 14 days, with status updates if it takes longer.

## Enforcement

Per Contributor Covenant 2.1's Enforcement Guidelines, response actions range from a private clarification through to a permanent ban from project spaces. The enforcement team selects the response proportional to the violation, with a bias toward education on first incidents and toward firmer action on repeated or severe violations.

Decisions are recorded internally; the affected party is notified; the enforcement team conducts a brief retro after the case closes.

## Maintainers' commitment

The maintainers commit to:

- Treating reports seriously and promptly.
- Maintaining confidentiality where requested, to the extent compatible with effective enforcement.
- Recusing themselves when they have a conflict of interest.
- Updating these processes openly when they prove inadequate.
